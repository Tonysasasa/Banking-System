# Banking-system
This is a self-designed banking system based on JAVA and PostgreSQL DBMS 
-------------------------------------------------------------------------

Functionalities:
1. Login
2. Create new account
3. Deposit/Withdraw money from account
4. Transfer money between accounts 

* Account ID number is unique
-------------------------------------------------------------------------
Required tools:
1. JAVA
2. PostgreSQL
3. Postgres JDBC
